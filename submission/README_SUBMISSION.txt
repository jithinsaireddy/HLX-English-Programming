IEEE Submission Package Checklist

Manuscript
- main.tex (IEEEtran)
- refs.bib
- paper/table_evaluation.tex
- paper/figure_pipeline.tex
- paper/hlx_pipeline.pdf (figure)

Generated PDF
- main.pdf (compiled, fonts embedded; run through IEEE PDF eXpress after acceptance)

Metadata & Letters
- cover_letter.txt (this folder)
- reviewers.csv (suggested reviewers; no COI)

Notes
- Ensure all fonts embedded in PDF; figures are vector (PDF) where possible
- Verify no unresolved references (no "Table ??"/"Figure ??")
- Ensure acronyms expanded on first use (IoT, WoT/TD, RTOS)
- Use PDF eXpress identifier provided by the target journal/conference when camera-ready
- Complete IEEE eCopyright form during final submission
- ORCID ID will be required by many IEEE journals (add in portal)
